1. What is Azure Linux Agent?
   A: Azure Linux Agent is a software written in Python, and it manages Linux provisioning and VM interaction with the Windows Azure Fabric Controller.

2. Why name changed to Azure Linux Agent from Windows Azure Linux Agent?
   A: Microsoft changed the name of Windows Azure to Azure, so we did corresponding change for our project.
